# Coffee License Stamps

Copy-paste ready Coffee License stamps for different programming languages.

## Variables to Replace
- `[YEAR]` → 2025 (or current year)
- `[NAME]` → Your name or company name
- `[EMAIL]` → Your contact email
- `[AMOUNT]` → License fee (default: $50)
- `[BIG_CORP]` → Example mega-corp (Google, Apple, Microsoft, etc.)

---

```javascript
/*
┌─────────────────────────────────────────────────────────────┐
│                     THE COFFEE LICENSE                      │
├─────────────────────────────────────────────────────────────┤
│ • Free for everyone.                                        │
│ • Coffee money for billionaires.                            │
│                                                             │
│ MIT + Basic Courtesy                                        │

│ • Free for personal, educational, and small business use    │
│ • $[AMOUNT] if your net worth has commas and lawyers        │
│ • Use freely, modify freely                                 │
│   (but buy me a damn latte if you're [BIG_CORP])           │
├─────────────────────────────────────────────────────────────┤
│ Copyright [YEAR] [NAME]                                     │
│ Contact: [EMAIL]                                            │
├─────────────────────────────────────────────────────────────┤
│ IF YOU'RE ABOVE THE THRESHOLD                              │
│ but would prefer not to pay,                               │
│ restructure your assets                                    │

│ ☕ Made with code and good faith                            │

└─────────────────────────────────────────────────────────────┘
*/
```
